# UI设计规范（板块文档抽取）

## 范围与目标
- 范围：经营概览、保费进度、变动成本、损失暴露、费用支出与筛选器系统
- 目标：统一视觉风格、提升可读性、保证交互一致性

## 全局UI规范
### 1. 图表容器与尺寸
- 图表容器：浅色玻璃质感背景、圆角、阴影（`chart-container`）
- 图表高度：默认 500px，宽度 100%
- 图表网格线：默认隐藏

### 2. 线条与轴样式
- 折线宽度：统一 1px
- 坐标轴线：1px 细线
- 坐标轴标签：加粗、优先显示全部，长文本可换行或截断

### 3. 数据标签与提示框
- 柱状图标签：柱内或柱顶显示，优先加粗以提高对比度
- 折线标签：数据点上方显示
- Tooltip：展示维度名 + 关键指标；当轴已标注单位时，提示框不重复单位

## 颜色系统（已在板块文档出现的配色）
### 1. 状态色
- 正常：`#00b050`
- 预警：`#ffc000`
- 危险：`#c00000`

### 2. 经营概览
- 满期赔付率：`#87ceeb`
- 费用率：`#d3d3d3`
- 折线（达成率/贡献度）：`#0066cc`

### 3. 费用支出
- 费用金额：`#808080`
- 费用率折线：`#0066cc`
- 费用超支额柱：`#d0d0d0`（正值/负值标签分别红/绿）
- 费用率超支线：`#0070c0`
- 投产效率：保费比 `#0070c0`、边际贡献 `#00b050`（负值红色）、边际贡献率 `#c00000`

### 4. 损失暴露
- 赔款贡献度：默认灰色；当赔款贡献度 > 保费贡献度时显示红色
- 满期赔付率折线：灰色连线，红色点

## 模块级图表形态与UI要点
### 1. 经营概览
- KPI卡片 + 堆积柱（满期赔付率+费用率）+ 折线
- 左右Y轴范围保持一致，避免视觉误导
- KPI卡片采用大号数值字体与状态色

### 2. 保费进度
- 单系列柱状图
- 柱色由变动成本率的三色状态决定
- Tooltip 展示维度与签单保费（万元）

### 3. 变动成本
- 双轴散点图（满期赔付率/费用率）
- 气泡大小代表保费占比，颜色为变动成本率状态色
- 预警线：满期赔付率与费用率虚线

### 4. 损失暴露
- 赔付率VS占比：双Y轴组合图（两组柱 + 折线）
- 频度VS额度：散点象限图，动态预警线（摩托车模式）
- 子标签切换（bubble/quadrant）

### 5. 费用支出
- 费用金额VS费用率：双Y轴组合图
- 费用超支分析：双Y轴组合图（超支额柱 + 费用率超支线）
- 投产效率分析：双Y轴组合图（两组柱 + 折线）

### 6. 筛选器系统
- 电商式下拉筛选器
- 搜索框 + 全选/反选/清空/确定操作区
- 锁定状态显示锁图标，按钮样式变化

## 交互规范
- 维度切换：按钮组单选切换，激活态高亮
- 子标签切换：用于损失暴露双模式
- Tooltip：跟随鼠标，展示完整指标信息
- 键盘与焦点：维度按钮与子标签可聚焦（保持可访问性）

## 响应式规则
- KPI卡片：移动端 1 列 / 平板 2 列 / 桌面 4 列
- 图表容器宽度自适应，保证标签可读性
