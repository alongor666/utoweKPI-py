# **第一部分：全局设计规范（Global Settings）**

## 0.1 图表基础样式规范

1. 去掉全部网格线
2. 所有文字使用粗体
3. 预警线清晰可见
4. X 轴文字不可倾斜
5. X 轴文字自动缩小适应间距
6. X 轴文字不允许交叉
7. **气泡图坐标轴**：X轴和Y轴名称必须清晰可见，不得被遮挡
8. 全部图表必须排序（最差 → 最好）
9. 气泡图气泡大小统一为：**变动成本率**
10. 气泡图需添加标签：**机构名称 + 关键指标值**（避免仅悬停可见）
11. **移除图表独立标题**：因每个PPT页面已有主标题，图表自身不再展示标题
12. **页面主体框架**：采用 16:9 PPT 幻灯片比例，白色背景，圆角阴影。

## 0.2 排序规范

1. 从最差排序到最好
2. 异常程度越高越靠前
3. 排序维度优先级：三级机构 → 客户类别 → 业务类型
4. 图表排序与标题排序保持一致

## 0.3 标题规范

1. 每个标签页必须配置标题
2. 标题左对齐
3. 标题颜色与项目主色一致
4. 标题遵循排序逻辑
5. 三级机构标题不展示“本部”
6. **标签页命名统一**：统一使用“三级机构”、“客户类别”、“业务类型”，禁止使用“按”字前缀
7. **统一格式**：`标签名（子标签）——标题名：范围内的机构列表`
   * 示例：`损失暴露（气泡图分析）——高赔付高占比机构：宜宾、德阳`
8. **项目主标题**：`XX分公司车险第N周经营分析`
9. **数据截止日期**：`数据截止日期：YYYY年MM月DD日`（日期为该周对应的星期六）

## 0.4 气泡图规范**

1. 气泡大小 = **变动成本率**
2. 气泡大小需标注对应指标
3. 气泡比例固定
4. 坐标轴范围全局一致

## **0.5 三级机构展示规范**

1. 全部图表不展示本部
2. 仅展示 12 家三级机构（按排序逻辑前12家）

## 0.6 正文规范

1. 正文按固定分类列明
2. 正文使用粗体、左对齐
3. **正文序号**：必须使用符合国际规范的序号格式（如 `1.`, `2.`），后接空格
4. 仅出现分类名称
5. 不出现解释、原因、分析、建议
6. 一字不差使用标准命名
7. 正文段落数量与分类完全对应
8. **正文必须列出对应的机构/类别名称**
   * 示例：`1. 高占比低赔付机构：天府、武侯`

## 0.7 阈值设定

1. **出险率阈值**：12%
2. **案均赔款阈值**：4800元
3. **KPI 状态颜色及阈值**：
   * **保费时间进度达成率**：
     * < 95%：红色（status-danger）
     * 95% - 100%：黄色（status-warning）
     * ≥ 100%：绿色（status-good）
   * **满期赔付率**：
     * > 75%：红色（status-danger）
     * 70% - 75%：黄色（status-warning）
     * ≤ 70%：绿色（status-good）
   * **费用率**：
     * > 17%：红色（status-danger）
     * 14% - 17%：黄色（status-warning）
     * ≤ 14%：绿色（status-good）
